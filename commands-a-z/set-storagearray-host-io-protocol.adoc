= 스토리지 어레이 호스트 I/O 프로토콜 설정 - SANtricity CLI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
 `set storageArray hostIoProtocol` 명령은 스토리지 어레이 수준에서 호스트 I/O 프로토콜을 구성합니다.



== 지원되는 어레이

이 명령은 모든 SMcli 패키지가 설치되어 있는 한 E4000, E2700, E5600, E2800, E5700, EF600, EF300, EF50 및 EF80 어레이를 포함한 모든 개별 스토리지 어레이에 적용됩니다.



== 역할

E4000, E2800, E5700, EF600, EF300, EF50 또는 EF80 스토리지 어레이에서 이 명령을 실행하려면 Support Admin 역할이 있어야 합니다.



== 구문

[source, cli, subs="+macros"]
----
set storageArray hostIoProtocol={"FC" | "ISCSI" | "ISER" | "nvmeIb" | "nvmeFc" | "nvmeRoce" | "SAS" | "SRP"}
----


== 매개 변수

[cols="2*"]
|===
| 매개 변수 | 설명 


 a| 
`set storageArray hostIoProtocol`
 a| 
스토리지 어레이에 사용할 호스트 I/O 프로토콜을 지정할 수 있습니다. 유효한 선택 사항은 FC, ISCSI, ISER, nvmeIb, nvmeFc, nvmeRoce, SAS 또는 SRP입니다.

|===
